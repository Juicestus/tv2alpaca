TradingView 2 Alpaca


TradingView 2 Alpaca (tv2alpaca) is a Webhooks bridge to convert TradingView Webhooks to Alpaca Trade API orders.
See the code on GitHub.

How to Use


• Step 1: Get your Alpaca API Key ID and Secret Key of your paper or live trading account from the Alpaca Dashboard.

• Step 2: Setup the TradingView Webhook Alerts in your TradingView chart.

• Step 3: In the URL box, enter:
https://www.tv2alpaca.com/api/{your key id here}/{your secret key here}/endpoint

• Step 4: In the data box, enter:
{{fmrt}}
